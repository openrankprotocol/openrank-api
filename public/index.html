üìñ Query Parameters

For /scores endpoints, use these optional query parameters:

  • start - Starting index for pagination (default: 0)
  • size - Number of items to return (default: all)

üíª Example Usage

Get complete Discord dataset:

curl https://your-deployment.vercel.app/api/discord/ritual

Get GitHub seed data:

curl https://your-deployment.vercel.app/api/github/bitcoin/seed

Get paginated scores:

curl "https://your-deployment.vercel.app/api/discord/ritual/scores?start=0&size=10"

Get X community ID:

curl https://your-deployment.vercel.app/api/x/ritual-community/community_id

üîß Response Format

All responses are returned in JSON format. Score entries follow this structure:

  "i": "identifier",
  "v": 123.45
